The section tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document.

The footer tag defines a footer for a document or section. A footer element should contain information about its containing element. A footer element typically contains: authorship information copyright information contact information sitemap back to top links related documents You can have several footer elements in one document.
